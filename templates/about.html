Library Management System


“Library Management System” for monitoring & updating the activities happened in the Library. This project “Library Management System” is developed in Python-Django & we use PostgreSQL for storing the our databases. This project mainly focuses on the basic activities in the Library like adding new books, updating information’s regrading issuing or submission of books, searching books or students.

Using HTML, CSS, and JavaScript, we will design a GUI for library management system where Students can logged-in to access its features.

Throughout this project we are focusing on the presenting information in an easy and efficient manner.
